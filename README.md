Shell-ish Shell

A simple linux shell implementation was done. It had some previously built in commands such as cd and exit.
I futher added the implementtion of cut and a custom command called 'recent'. Moreover, the input/output files' redirections 
and background execution of programs can be handled in the shell. 



The logic behind the custom command is that it shows the 3 (default number) most recently visited files in the chosen 
directory. For this, we have to run the shell first, then type "recent <directory name>". We will be presented the 3 visited
files sorted by their modification time.
